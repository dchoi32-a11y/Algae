  • Temperature: 25C

    All samples will be in the same room. This is for a fair experiment, and since algae development differs in temperature.

  • Volume of the water: 100ml

    Measure with a graduated cylinder. Different volumes weaken nutrients differently.

  • Electronic Scale

    Use the same electronic scale each time. Not using the same one when uncablibrated will make results inaccurate.

  • Size of container

    Use identical petri dishes. Surface area affects light exposure.
